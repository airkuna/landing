## Stil i pravila pisanja
- **Jednostavan rječnik.** Svaka rečenica razumljiva osobi bez ekonomskog obrazovanja.
- **Bez filozofiranja.** Problem → broj → posljedica → rješenje.
- **Bez emojija** (brand pravilo iz pitch-deck repoa).
- **Svaki broj ima izvor** (inline ili u `11-izvori.md`).
- Pojmovi (EMT, mint, burn, SEPA, MoR) prvi put → kratko objašnjenje ili link na `12-pojmovnik.md`.
